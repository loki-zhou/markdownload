MarkDownload 边栏功能测试

功能说明

这是一个测试页面，用于验证 MarkDownload 扩展的新边栏功能。

主要改进

  • 边栏显示：扩展现在以边栏形式显示，而不是小弹窗
  • 更大空间：边栏提供更多空间来预览和编辑 Markdown 内容
  • 持久化：边栏不会因为点击其他地方而关闭
  • 更好体验：可以在浏览网页的同时查看转换结果

使用方法

  1. 点击浏览器工具栏中的 MarkDownload 图标
  2. 边栏将在右侧打开
  3. 选择要转换的内容类型（选中文本或整个文档）
  4. 在边栏中预览 Markdown 结果
  5. 点击下载按钮保存文件

技术实现

使用了 Chrome 扩展的 sidePanel API，这是 Manifest V3 中的新功能。边栏具有以下特点：

  • 响应式设计，适配不同宽度
  • 保持原有的所有功能
  • 优化的用户界面布局
  • 支持深色模式

这段文字可以用来测试选中文本的转换功能。选中这段文字，然后在边栏中选择"选中文本"选项。

代码示例

以下是一个简单的 JavaScript 代码示例：

// 边栏打开处理
chrome.action.onClicked.addListener(async (tab) => {
  await chrome.sidePanel.open({ tabId: tab.id });
});
